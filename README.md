## ğŸ”§ Customization / ĞĞ°ÑÑ‚Ñ€Ğ¾Ğ¹ĞºĞ°

### Changing Content
- Edit text directly in `index.html`
- Update translations in `assets/js/translations.js`

### Changing Images
- Replace files in `assets/images/` keeping the same filenames
- Recommended image size: 500x500px or larger (square format)

### Changing Video Background
- Replace `assets/video/video.mp4` with your video
- Supported format: MP4 (H.264 codec recommended)
- Recommended: Short loop video (15-30 seconds)

### Changing Documents
- Replace PDF files in `assets/documents/`
- Keep filenames or update paths in `index.html`

### Contact Form Setup
- The form uses Web3Forms API
- To change recipient, update `access_key` in the form's hidden input
- Register at [web3forms.com](https://web3forms.com) for your own API key
